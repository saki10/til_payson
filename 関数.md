# 関数
def 関数名（引数1,引数2）:
    return

* 関数は引数を受け取ることができる。
* 受け取った引数は、関数を使うことができる
* 引数は省略することができる
* 引数は何個も渡すことができる


def say_hello():
    print("hallo")


say_hello()
say_hello()
say_hello()

==>hallo
==>hallo
==>hallo


## 引数あり
def say_hello(greeting):
    print(greeting)

say_hello("hallo")
